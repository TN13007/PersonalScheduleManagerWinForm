# PersonalScheduleManagerWinForm
## 1. Mô tả

**PersonalScheduleManagerWinForm** là phần mềm quản lý thời gian biểu cá nhân chạy trên winform, cho phép người dùng tạo, theo dõi và điều chỉnh lịch trình công việc theo thời gian.

Hệ thống được thiết kế theo nguyên lý lập trình hướng đối tượng (OOP) và áp dụng các design pattern:

- Inheritance
- Polymorphism
- Abstraction
- Builder Pattern
- Repository Pattern

Dữ liệu được lưu trữ bằng cơ chế serialization/deserialization ra file JSON.

## 2. Chức năng chính
- Hiển thị danh sách công việc
- Hiển thị:
  - Tiêu đề
  - Thời gian bắt đầu
  - Thời gian kết thúc
  - Nội dung công việc

- Thêm công việc  
- Chỉnh sửa công việc  
- Xóa công việc  
- Xem danh sách  

- Lưu dữ liệu ra file JSON  
- Đọc dữ liệu từ file JSON  

- Sắp xếp tìm kiếm công việc
